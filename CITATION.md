# How to cite

If you use ProteinGymPy in a publication, please cite:

Tyrone Lee and Tram Nguyen (2025). ProteinGymPy v0.1.0: Programmatic access to ProteinGym datasets with visualization tools. GitHub repository, https://github.com/<your-org-or-username>/ProteinGymPy. DOI: 10.xxxx/zenodo.xxxxx

@software{proteingympy2025,
  author = {Lee, Tyrone and Nguyen, Tram},

  year = {2025},
  version = {0.1.0},
